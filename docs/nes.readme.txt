= AllyWorker Pro =

[AllyWorker Pro](https://allyworker.com/pro/) extends the free plugin with the most powerful abilities for development and staging environments:

* **Execute PHP** inside the live WordPress process — full access to `$wpdb`, all functions, all active plugins
* **Run WP-CLI commands** — plugin management, post operations, cache flushes, database imports, anything WP-CLI supports
* **Query the database** via parameterised `$wpdb` SQL
* **Write files** anywhere on the server, with automatic `.bak` backups before overwrites
* **Edit files** with targeted find-and-replace (safer than a full overwrite)
* **Delete files and directories**, with protection for core WordPress paths

Pro abilities appear in **AllyWorker → Ability Settings** alongside free abilities and can be enabled or disabled individually.
