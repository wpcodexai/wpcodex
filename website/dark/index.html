Capabilities

Everything your AI agent needs

A complete set of abilities so your AI can read, write, and execute anything on your WordPress site — with no sandbox limitations.

⚡

PHP Execution

Run arbitrary PHP code inside the WordPress process. Access all globals, hooks, functions, and database abstractions exactly as WordPress does.

🖥️

WP-CLI Access

Execute any WP-CLI command — install plugins, manage users, export data, run cron, and more. Full shell power, no SSH required.

🗄️

Database Queries

Run raw SQL via $wpdb. SELECT returns structured rows; mutations return affected count. Full access to every table.

📁

Filesystem Access

Read and write any file on the server. Atomic writes with automatic .bak backups keep your data safe during edits.

⚙️

Options API

Get and set any WordPress option directly. Configure plugins, themes, and core settings without navigating admin panels.

📝

Post Queries

Run WP_Query with any args. Returns found_posts count plus a full posts array — ready for the AI to analyse or modify.

🧠

Skills System

Store AI playbooks in the database. Agents load skills at session start for standing instructions specific to your site's setup.

🔐

Secure by Default

The MCP server is disabled by default. It only activates when you add a single constant to wp-config.php. No surprise exposure.

🔌

Extensible

Register custom abilities with a single function call. Filter the abilities list to add your own tools or integrate third-party APIs.

Skills System

Teach your agent your site's patterns

Skills are Markdown playbooks stored in the database. Create one after solving a tricky problem — the agent loads it at session start every time.

  • 📚

    Persistent site memory

    Skills remember your theme IDs, plugin configs, and custom post types so agents don't rediscover them.

  • ⚡

    Instant context loading

    Agentic skills load automatically. Task-specific skills load on demand. Zero setup per session.

  • ✏️

    Easy to create and edit

    Create skills in the WordPress admin or let the agent write them after completing a complex task.
