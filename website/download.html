System requirements

Required

  • ✓ WordPress 6.9 or higher
  • ✓ PHP 8.0 or higher
  • ✓ WordPress Application Passwords enabled (default in WP 5.6+)
  • ✓ REST API accessible (not blocked by security plugins)

Recommended

  • → PHP 8.2+ for best performance
  • → WP-CLI installed on the server
  • → HTTPS / TLS certificate
  • → Dedicated admin account for agent access

After downloading

1

Upload and activate

Upload the ZIP via Plugins → Add New → Upload Plugin, or unzip into /wp-content/plugins/wpworker/. Activate the plugin.

2

Enable MCP

Add define('WPCODEX_ENABLE_MCP', true); to your wp-config.php before the "That's all" line.

3

Create an Application Password

Go to Users → Profile → Application Passwords and generate one for your AI agent.

4

Connect your agent

Point your MCP client at https://yoursite.com/wp-json/wpworker/v1/mcp using Basic auth. See the Quick Start guide for client-specific instructions.
